Rugix Admin exposes an HTTP API used by its web interface.

::::danger[Unstable API]

This API is considered an implementation detail of Rugix Admin. We do not
recommend building external tooling or integrations on top of it yet. **The API may change
at any time, including in breaking ways, without the compatibility guarantees that apply
to documented Rugix user interfaces.**

::::
